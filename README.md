# demo-evento-gastronimico
Landing inmersiva para cenas exclusivas o restaurantes con estrella Michelin. Incorpora efecto Parallax CSS nativo sin JS, menú degustación maquetado con estilo clásico editorial, y formulario de reservas orientado a transacción y control de alérgenos. Paleta vino, carbón y cobre.
